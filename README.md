# aedin-insight

### Sovereign Cognitive Infrastructure Blueprints

The production codebase and web repository for **The Automated Enterprise** series. This system functions as an open-source technical asset library, mapping out high-density architectural blueprints for mid-market operators looking to deploy autonomous intelligence without linear headcount growth.

---

## 🛠️ System Architecture

Built from a mobile-first engineering perspective (Termux/Ubuntu 24.04 LTS), this deployment framework utilizes a zero-bloat static distribution methodology:

*   **Substrate:** Pure HTML5 / Tailwind CSS Engine
*   **Routing Matrix:** Isolated `.nojekyll` pipeline for raw web indexing
*   **Target Infrastructure:** GitHub Pages Static Edge Network
*   **Custom Apex Engine:** Sovereign domain mapping (`aedininsight.com` / `aedininsight.info`)
